# 📘 Backend Course

---

## 📦 Installation and launch
```bash
git clone https://github.com/edward-tobilko/backend-course.git
cd backend-course
yarn install
```

### Development mode
```bash
yarn dev
```

### Debug-mode
```bash
yarn debug:build
```

### Compilation and production
```bash
yarn build
yarn start
```

### Tests
```bash
yarn test
yarn test:e2e
```

#### The server starts by default -> http://localhost:5001
or 
#### "Render" hosting -> https://backend-course-0wnd.onrender.com

### ⚙️ Tech Stack
- Express
- Node >= 20v
- TypeScript
- cross-env + TSX
- rimraf
- dotenv
- zod
- Jest
